## 기술스택

### Spring

- Java 11
- Gradle 7.2
- Spring Boot 2.6.0
- Spring Data JPA, QueryDSL
- Spring Security, jwt
- JUnti5, RestDocs, JACOCO

### Database

- H2
- MySQL

### DevOps

- AWS - EC2 / RDS / S3 / CloudFront
- Github Actions
- Nginx
- Docker, docker-compose
- Slack, Codecov
